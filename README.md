# Studio Ghibli API Integration Task

This project was made for [Ghibli API](https://ghibliapi.vercel.app/) integration using [RTK](https://redux-toolkit.js.org/)
and [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)
with [Jest](https://jestjs.io/).
The project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Features

- List all Studio Ghibli films.
- View characters (people) for each film.
- Select a film to see its characters in a table.
- Responsive navigation sidebar.
- "Read More" expandable description for long text.
- Accessibility friendly.
- Unit and integration tests for components.

## Project Setup

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the development server.
